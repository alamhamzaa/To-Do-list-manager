# 📝 Pro To-Do List Manager (ES6 Project)

## 📌 Overview

This is an advanced **To-Do List Manager** built using **JavaScript ES6**, designed to simulate a real-world task management application.
It supports task creation, editing, deletion, filtering, analytics, and user-based data storage.

---

## 🚀 Features

### ✅ Core Features

* Add, edit, and delete tasks
* Mark tasks as completed or pending
* Filter tasks (All / Completed / Pending)
* Persistent storage using **localStorage**

### 🔥 Advanced Features

* 🔐 User Login System (localStorage-based)
* 📊 Task Analytics Dashboard (Total, Completed, Pending, %)
* 🔍 Real-time Search Functionality
* 🏷️ Priority Levels (High / Medium / Low)
* 🌙 Dark Mode (Saved preference)
* 🧠 Dynamic DOM Manipulation
* 💾 User-specific data storage
